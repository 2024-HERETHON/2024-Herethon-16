- **개발환경에서의 실행 방법**
  ```
  $ python -m venv newvenv    #가상환경 생성
  $ source newvenv/Scripts/activate
  $ pip install django
  $ python -m pip install Pillow # 이미지 처리를 위한 라이브러리 설치
  $ cd danjang_prj
  $ python manage.py runserver
  ```
